How to generate charts

Stage 1: x-Axis and y-Axis Label Entity Recognition

Neural Networks were built which were trained and fed with representations of words in a way that a computer can understand. Seperate models are built to identify the X axis - referred to as the "independent variable" which are categorical and the Y axis(dependent variable) which is numerical.




The architecture above shows how data is "inputted" to the model and how it is processed. Based on the data, it will make a prediction or result, which is known as an "output" Between this, the model uses Bi-directional LTSM networks. It is capable of learning long-term dependencies in data sequences which allows it to retain certain information for longer. The data is then passed to the time-distribution layer and dense layer to classify each word index that falls into a category of a respected entity or not.
